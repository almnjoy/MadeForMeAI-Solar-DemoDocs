Your AI runs on a large language model under the hood. You can switch which model it uses directly from the chat.

## How to switch models

Look for the model selector dropdown at the top of the chat interface. Click it to see available models and select one.

## Which model should I use?

**Standard and Power plans** come with Claude Sonnet pre-configured — a strong, well-rounded model that handles most tasks well. Most users never need to change it.

**BYOK plan** defaults to a fast, budget-friendly model. You can switch to a more powerful one after [adding your API key](/byok/adding-your-key).

## Tips

- Faster models are good for quick tasks (drafting, answering questions)
- More capable models handle complex analysis, long documents, and nuanced writing better
- If a model isn't responding, try switching to a different one
