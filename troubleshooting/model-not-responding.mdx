When your AI assistant stops responding mid-conversation or won't reply at all, it's usually caused by a temporary model hiccup, an API key issue, or an outage from the underlying AI provider. Work through the steps below to get things moving again.

<Steps>
  <Step title="Refresh the page">
    Start with the simplest fix. Refresh your browser tab and send your message again. A brief network interruption is often all it takes to stall a response, and a refresh clears it immediately.
  </Step>

  <Step title="Switch to a different model">
    Use the model picker dropdown at the top of the chat to select a different model. If the model you're currently using has an outage or a temporary issue, switching to another one will get you a response right away.
  </Step>

  <Step title="Check your API key (BYOK plan only)">
    If you're on the BYOK (Bring Your Own Key) plan, a misconfigured or depleted API key is the most likely cause.

    1. Go to **Settings → API Keys** and confirm your key is entered correctly — no extra spaces, no missing characters.
    2. Log in to your account with the AI provider (OpenAI, Anthropic, or OpenRouter) and check that you still have credits or quota available.
    3. If the key looks fine, try switching to a different model from the same provider.

    <Note>
      If you're not on the BYOK plan, your API access is managed by MadeForMeAI and you can skip this step.
    </Note>
  </Step>

  <Step title="Check for provider outages">
    If no model responds and your API key is fine, the AI provider itself may be experiencing an outage. Check the status pages for the relevant providers:

    - [status.openai.com](https://status.openai.com)
    - [status.anthropic.com](https://status.anthropic.com)
    - [openrouter.ai](https://openrouter.ai) — check their Discord or status page

    If a provider is reporting an incident, the best approach is to wait for them to resolve it and try again.
  </Step>
</Steps>

## Still not working?

If you've been through all of the steps above and your AI still isn't responding, reach out to the MadeForMeAI team.

- **Discord:** [discord.gg/madeformeai](https://discord.gg/madeformeai) — the fastest way to get help
- **Email:** support@madeformeai.com

<Info>
  When you contact support, include your subdomain name and a description of what you're seeing (for example, which model you're using and whether you get any error message). This helps the team check the logs and diagnose the issue faster.
</Info>
