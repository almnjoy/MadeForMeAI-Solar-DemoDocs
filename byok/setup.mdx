BYOK (Bring Your Own Key) is for people who already have an API key from OpenAI, Anthropic, or OpenRouter. You supply the key, we run the private infrastructure.

## When BYOK makes sense

- You already pay for an OpenAI or Anthropic subscription and have API access
- You want to use a specific model that you're already paying for elsewhere
- You're cost-conscious and want full control over your AI usage spend

## Supported providers

- **OpenRouter** — recommended, gives you access to hundreds of models with a single key
- **OpenAI** — GPT-4, GPT-4o, and others
- **Anthropic** — Claude models

## How to add your API key

<Steps>
  <Step title="Log in to your dashboard">
    Go to [madeformeai.com/dashboard](https://madeformeai.com/dashboard) and log in.
  </Step>

  <Step title="Find your API key section">
    On the dashboard, look for the **API Key** card. This shows your current key (masked).
  </Step>

  <Step title="Rotate or replace the key">
    Click **Rotate key** to update it. Paste your new API key from your provider.
  </Step>
</Steps>

## Where to get an API key

**OpenRouter** (recommended for BYOK): Sign up at [openrouter.ai](https://openrouter.ai). Create an API key in your account settings. You can deposit credits and use them across hundreds of models.

**Anthropic**: Sign up at [console.anthropic.com](https://console.anthropic.com). API keys are in Account Settings.

**OpenAI**: Sign up at [platform.openai.com](https://platform.openai.com). API keys are in the API section.

## Usage costs

With BYOK, you pay your AI provider directly for what you use. Costs vary by model and usage volume. Check your provider's pricing page for details.

Your $15/month MadeForMeAI subscription covers the platform — the private server, memory, app connections, and everything else.
