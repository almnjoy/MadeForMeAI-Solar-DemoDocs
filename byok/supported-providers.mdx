---
title: Supported Providers
description: Which API key providers work with BYOK.
---

MadeForMeAI supports three API key providers for BYOK users.

## OpenRouter (recommended)

**Variable:** `OPENROUTER_API_KEY`
**Get a key:** [openrouter.ai](https://openrouter.ai)

OpenRouter gives you access to models from OpenAI, Anthropic, Google, Meta, and more — all with a single key. You only pay for what you use, with no monthly minimums.

This is the best option for most BYOK users because you get the widest model selection and can switch between providers without getting a new key.

---

## OpenAI

**Variable:** `OPENAI_API_KEY`
**Get a key:** [platform.openai.com](https://platform.openai.com)

Gives you access to GPT-4o, GPT-4o mini, and other OpenAI models. You'll need to add credits to your OpenAI account before use.

---

## Anthropic

**Variable:** `ANTHROPIC_API_KEY`
**Get a key:** [console.anthropic.com](https://console.anthropic.com)

Gives you access to Claude models (Sonnet, Haiku, Opus). You'll need to add credits to your Anthropic account before use.

---

## Which should I pick?

If you're not sure, go with **OpenRouter**. One key, all models, pay as you go. It's the most flexible option with the lowest barrier to entry.
