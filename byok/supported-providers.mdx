MadeForMeAI supports three AI providers for BYOK: OpenRouter, OpenAI, and Anthropic. Each gives you access to different models and has its own billing model. You only need one key to get started — read through the options below to find the right fit for you.

## OpenRouter (recommended)

**Environment variable:** `OPENROUTER_API_KEY`

OpenRouter is a unified API gateway that routes requests to models from OpenAI, Anthropic, Google, Meta, and many other providers — all through a single key. You pay only for what you use, with no monthly minimums.

This is the best option for most BYOK users. You get the widest model selection available, and you can switch between providers at any time without generating a new key.

**How to get a key:**
1. Go to [openrouter.ai](https://openrouter.ai) and create an account.
2. Navigate to **Keys** in your dashboard.
3. Generate a new API key and copy it.

---

## OpenAI

**Environment variable:** `OPENAI_API_KEY`

An OpenAI key gives you direct access to GPT-4o, GPT-4o mini, and other OpenAI models. You'll need to add credits to your OpenAI account before your key can be used for API calls.

**How to get a key:**
1. Go to [platform.openai.com](https://platform.openai.com) and sign in or create an account.
2. Navigate to **API Keys** in the dashboard.
3. Click **Create new secret key**, copy it, and store it somewhere safe — you won't be able to view it again.
4. Add credits to your account under **Billing** if you haven't already.

---

## Anthropic

**Environment variable:** `ANTHROPIC_API_KEY`

An Anthropic key gives you direct access to the Claude family of models, including Claude Sonnet, Haiku, and Opus. Like OpenAI, you'll need to add credits to your Anthropic account before use.

**How to get a key:**
1. Go to [console.anthropic.com](https://console.anthropic.com) and sign in or create an account.
2. Navigate to **API Keys**.
3. Create a new key, copy it, and store it somewhere safe.
4. Add credits to your account under **Billing** before making API calls.

---

## Which provider should you choose?

<Tip>
If you're unsure, start with **OpenRouter**. One key gives you access to all major model families, you pay only for what you use, and there are no monthly minimums. It's the lowest-friction way to get started with BYOK.
</Tip>

If you already have an existing OpenAI or Anthropic account with credits, you can use that key directly — there's no need to create a new account just for BYOK.

Once you've chosen a provider, follow the instructions in [Add Your API Key](/byok/adding-your-key) to connect it to your MadeForMeAI instance.
