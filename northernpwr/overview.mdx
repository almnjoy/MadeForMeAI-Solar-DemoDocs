# NorthernPWR AI Demo

This section covers the NorthernPWR AI agent demo built on MadeForMeAI.

NorthernPWR is a Canadian residential solar installer operating across Alberta, British Columbia, Manitoba, and Saskatchewan. The demo shows three AI agents running in their Discord server, each with a distinct purpose.

## The Three Agents

| Agent | Channel | Purpose |
|-------|---------|---------|
| **Solar Doc** | #solar-info | Answers questions about how solar works, components, and the install process |
| **Solar Support** | #solar-support | Helps customers troubleshoot, navigate warranties, and get connected with NPWR |
| **Solar Sales** | #solar-sales | Handles inbound interest, qualifies leads, guides prospects toward a free quote |

Each agent is a separate Hermes profile with its own Discord bot, persona, and knowledge base. They hand off to each other using channel redirects.
